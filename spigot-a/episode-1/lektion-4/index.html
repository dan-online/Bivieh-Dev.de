Episode 1, Lektion 4

Um alle Plugins, die wir gemeinsam programmieren werden, testen zu können, erstellen wir uns unseren eigenen Server auf dem PC. Wir nutzen dabei die Serversoftware Spigot auf der Version 1.8.

Übungen

Start Script

Start.bat-Skript: ​ java -Xmx1G -jar spigot.jar pause

Was ist Spigot?

Spigot ist eine Serversoftware, die auf CraftBukkit basiert. Verschiedene Features des Vorgängers wurden optimiert um ein optimales Nutzererlebnis zu ermöglichen. Wir nutzen Spigot zum Einen um unseren Server zum Laufen zu bekommen, zum Anderen aber auch als Basis zur Programmierung unserer Plugins.
